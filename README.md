Online Quiz Application – Project Description

Online Quiz Application is a full-stack web-based application developed to provide an interactive platform for users to participate in online quizzes. The application allows users to register, log in, attempt multiple-choice quizzes, and view their scores. It provides an efficient way to manage quiz questions and evaluate user performance.

The backend of the application is developed using Java and Spring Boot, which provides RESTful APIs for handling user authentication, quiz management, question retrieval, and score calculation. The APIs are tested and validated using Postman. The frontend is designed using HTML and CSS to provide a simple and user-friendly interface. The project uses Maven for dependency management and project build automation.
